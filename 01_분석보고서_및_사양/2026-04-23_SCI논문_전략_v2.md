## 1. SOTA 재조사 핵심 (2025–2026 Q1)

### 1.1 이미 포화된 영역 (≒ 추가 기여 작은 영역)

다음 분야는 2024–2025 동안 변종 논문이 쏟아져 단순 재조합으로는 mid-tier도 어렵다.

- **Dual EKF/UKF 변형군**: DEKF, DUKF, MIAUKF, GMCC-STASRUKF, dual H-infinity, dual fractional-order AUKF 등이 2025년에만 십수 편 ([MDPI 2025](https://www.mdpi.com/2673-3951/6/3/100), [ACS Omega](https://pubs.acs.org/doi/10.1021/acsomega.3c00918), [Nature SciRep 2025](https://www.nature.com/articles/s41598-025-24571-x), [IET Power Electronics 2025](https://ietresearch.onlinelibrary.wiley.com/doi/full/10.1049/pel2.70145), [J. Power Electronics 2025](https://link.springer.com/article/10.1007/s43236-025-01064-8)).
- **Multi-time scale fast/slow filter**: SVDUKF + EKF 조합은 [MDPI Batteries 2025](https://www.mdpi.com/2313-0105/11/11/410), [MDPI Batteries 2025-fractional](https://www.mdpi.com/2313-0105/11/10/372) 등이 이미 점령.
- **Aging-aware OCV mapping + EKF**: [Springer J. Power Electron. 2025](https://link.springer.com/article/10.1007/s43236-025-01064-8) — 본 프로젝트 Spec v1 §1.2 contribution과 거의 동일.
- **Transformer-based SOC/SOH 단일·다중 헤드**: [MDPI Energies 2024 Survey](https://www.mdpi.com/1996-1073/17/14/3502), [AIP Advances 2026 multi-task + cross-attention](https://pubs.aip.org/aip/adv/article/16/4/045009/3386119/) — 2026년 4월에 정확히 multi-task transformer + cross-attention 논문이 게재됨. 단순 multi-task TCN/Transformer 기여 약화.

→ **결론**: Spec v1의 Stage 1(Dual AEKF) 단독, Stage 2(Multi-task TCN) 단독으로는 reviewer가 "incremental"로 reject 할 위험 높음.

### 1.2 아직 비어 있는 영역 (white space)

다음 조합·각도는 2026-04-23 기준 게재 사례가 희소하거나 없음 — **여기를 노린다**.

1. **Joint(동시) PINN**: SOH 단독 PINN은 활발 ([ScienceDirect 2025 TSPINN](https://www.sciencedirect.com/science/article/abs/pii/S2095495625001214), [Nat. Commun. 2024 PINN-degradation](https://www.nature.com/articles/s41467-024-48779-z), [MDPI Batteries 2025](https://www.mdpi.com/2313-0105/11/2/49), [GitHub PINN4SOH](https://github.com/wang-fujin/PINN4SOH))이지만, **SOC와 SOH를 한 PINN으로 동시에 추정하면서 ECM 잔차를 물리 loss로 삽입하는 구조**는 공개 사례가 거의 없음. SOC PINN ([arXiv 2024](https://arxiv.org/html/2412.16724v1), [IEEE 2025](https://ieeexplore.ieee.org/abstract/document/10860002))과 SOH PINN을 한 네트워크로 병합하는 작업은 white space.
2. **Neural-Augmented Adaptive Kalman**: filter 구조 안에 잔차 보정 NN을 박는 하이브리드. Stage 3(stretch)였으나 격상 권고. PINN의 안정성 문제와 순수 DL의 일반화 문제를 동시 해결하는 컨셉으로 SCI 상위 핏.
3. **광범위 사이클 + 다온도·다채널 외삽 벤치마크**: 에기연 데이터는 단일 chemistry(NCM Pouch)로 25 °C × 0~3000 cycle, 50 °C × 0~3800 cycle, 0 °C × 0~100 cycle 의 사이클·온도 coverage를 가짐. 추가로 pouch_SOH 다채널(ch1~ch8) × 25/40 °C × 3종 패턴(MG/PVSmoothing/PowerQuality)으로 multi-cell 일반화 보강 가능. **"unseen aging × unseen temperature × unseen cell" 외삽을 정량화하는 벤치마크 자체를 contribution화**할 여지.
4. **Bayesian/Conformal 불확실성 정량화(UQ) 동반**: 대부분의 joint estimator 논문이 점추정에 머묾. PICP/NLL 같은 UQ 메트릭과 안전 마진(Safe SOC envelope)을 함께 보고하면 차별화.
5. **분수계 ECM(FOM) + 적응 Kalman의 Coupling을 PINN loss로 enforce**: 2025년 분수계 dual filter는 많지만, FOM 잔차를 NN의 부드러운 제약으로 쓴 hybrid는 희소.

### 1.3 본 프로젝트의 잔존 강점

조사 결과 다음은 여전히 본 프로젝트의 **고유 자산**이다.

- **JH3(Pouch 63 Ah, NCM 확정) + 광범위 사이클·다채널·다패턴 셀 데이터 + 동일 chemistry 14S 모듈 데이터**: 공개 데이터셋(NASA 18650 / CALCE 작은 셀 / MIT LFP)에는 이런 Pouch 폼팩터·다채널·다패턴·셀+모듈 동시 보유 조합이 없다.
- **RPT/DCIR/ACIR/Pattern 4종 동시 보유**: HI 추출 후보가 풍부.
- **단일 chemistry라는 약점**은 NASA/CALCE/MIT 다중 셀로 일반화 보강 가능.

## 2. 재정의된 Novelty (미드티어 노선 v2.1)

**노선 결정 (사용자 2026-04-23)**: 미드티어 SCI 허용 → 메인 contribution을 **PINN-AEKF에서 FOM-AEKF (Stage A)** 로 재격하. PINN은 stretch로 보존.

### 2.1 메인 Contribution Statement (Stage A — FOM-AEKF)

> We propose **FOM-AEKF**, a *Fractional-Order ECM coupled Dual Adaptive Extended Kalman Filter with Temperature-Aging scheduled Q/R*, for joint SOC/SOH estimation of NCM pouch lithium-ion cells **and their 14-series modules**. Unlike conventional dual AEKF approaches that adapt covariances solely from innovation residuals, the proposed method (i) replaces integer-order Thevenin with a fractional-order ECM (R0 + R1∥CPE) for higher voltage-fidelity under aged conditions, and (ii) introduces a **temperature × estimated-SOH 2D scheduling surface** for Q/R, learned offline from a thermally rich aging dataset (KIER NCM-pouch, 0/25/50 °C × 0~3,800 cycles). The same algorithm, trained on cell-level data, is further validated at the **module scale (14S NCM pouch module, ~3,000 cycles)** without re-training, demonstrating cross-scale generalization. Across the KIER main cells (ch2/ch7), pouch_SOH multi-channel side cells (ch1~ch8 with MG/PVSmoothing/PowerQuality patterns), and the 14S pouch module (CH1), FOM-AEKF achieves [X]% lower SOC RMSE under unseen aging and [Y]% lower SOH MAE compared to seven SOTA baselines (Ah / OCV / EKF / UKF / DEKF / DUKF / AEKF).

### 2.2 Stretch Contribution (Stage B/C — PINN-AEKF, P4 한정)

> Building on FOM-AEKF, we further inject a lightweight physics-informed neural residual with OCV–SOC and ECM consistency losses, plus an ensemble-based UQ head producing PICP_95 calibrated intervals.

### 2.3 3대 차별점 (Contribution bullets)

1. **Architectural**: 분수계 ECM + dual-time-scale AEKF + **온도×SOH 2D Q/R scheduling** 3-way 결합. 기존 dual AEKF 류는 innovation 잔차만 봄 → 본 연구는 환경·노화 의존성을 명시 모델링.
2. **Empirical (cell)**: NCM Pouch 단일 chemistry로 사이클 0~3,800, 온도 0/25/40/50 °C, 다채널(ch1~ch8) × 다패턴(MG/PVSmoothing/PowerQuality) 의 광범위 외삽 벤치마크 프로토콜 + 공개 코드.
3. **Empirical (cross-scale)**: 셀 학습 알고리즘이 **14S NCM Pouch 모듈에 추가 학습 없이 일반화** — Pouch 폼팩터 일관성을 유지한 cell→module 외삽은 SOC/SOH joint 분야에서 희소.
4. **Reliability (P4 stretch)**: 점추정에 머무르지 않고 calibrated UQ 동반 가능 — BMS 안전 마진 설계 직접 활용.

## 4. 알고리즘 세부 설계 — PINN-AEKF (P3 메인)

### 4.1 큰 그림

```
[V_t, I_t, T_t]
      │
      ▼
┌───────────────────────────────────┐
│  Fractional-Order ECM             │   physics block
│   V_pred = OCV(SOC,T) − I·R0      │
│           − Σ V_RC(α)             │
└───────────────────────────────────┘
      │ residual e_t = V_t − V_pred
      ▼
┌───────────────────────────────────┐
│  Dual-time-scale AEKF             │
│   • Fast: x = [SOC, V1, V2]       │
│   • Slow: θ = [R0, R1, Cn]        │
│   • Adaptive Q,R = f(T, SOH)      │
└───────────────────────────────────┘
      │ filter posterior
      ▼
┌───────────────────────────────────┐
│  Neural Residual Block (small)    │   data block
│   Δ = NN(e_t window, T, SOH)      │
│   → 보정된 SOC, SOH 출력           │
└───────────────────────────────────┘
      │
      ▼  Loss = L_data + λ1·L_phys(ECM 잔차) + λ2·L_OCV(SOC,V) + λ3·L_smooth(SOH)
```

### 4.2 손실 함수 (학습 단계에서)

```
L_total = L_data       (SOC, SOH 정답 vs 추정 MSE)
        + λ1 · L_phys  (ECM 단자전압 잔차)
        + λ2 · L_OCV   (SOC↔OCV 일관성)
        + λ3 · L_smooth (SOH의 시간미분 평탄성)
        + λ4 · L_calib (UQ calibration, P4에서 추가)
```

λ는 Kendall et al. (2018) **uncertainty-weighted** 방식으로 자동 균형.

### 4.3 추론 단계

학습 종료 후, 추론 단계에서는 NN이 매 스텝 실행되지만 파라미터는 고정. AEKF가 메인 필터로 동작하고 NN은 잔차 보정만 — 따라서 **BMS 임베디드 친화** (NN inference < 100 µs/step 목표).

### 4.4 분수계 ECM 도입 이유

- 정수계 1/2-RC는 EIS 데이터의 mid-frequency arc를 잘 못 맞춤.
- 분수계 (R + CPE) 모델은 same number of params로 fit이 좋음.
- 2025 SOTA 절반이 이미 FOM 채택 — 안 쓰면 비교에서 불리.

### 4.5 차별화 포인트 정리

| 요소 | 본 연구 | 2025 SOTA Dual AUKF | 2025 PINN4SOH | 2026 multi-task transformer |
|---|---|---|---|---|
| Fractional ECM | ✓ | 일부 | × | × |
| Adaptive Q/R w/ T,SOH | ✓ | innovation-only | × | × |
| Physics-informed loss | ✓ | × | ✓ (SOH only) | × |
| Joint SOC+SOH | ✓ | ✓ | × | ✓ |
| UQ (PICP/NLL) | ✓ | 드묾 | 드묾 | × |

## 5. 벤치마크 프로토콜

### 5.1 데이터셋 구성 (Pouch 폼팩터 일관, 두 축)

| 축 | 데이터셋 | 셀 종류 | 가용 그리드 | 역할 |
|---|---|---|---|---|
| 셀 | KIER main_cell | JH3 Pouch 63 Ah NCM | (ch2, 25 °C, 0~3000 cyc) + (ch7, 50 °C, 0~3800 cyc) + (ch3, 0 °C, 0~101 cyc 부분) | 메인 학습·검증 |
| 셀 | KIER pouch_SOH | NCM Pouch (다채널) | ch1~ch8 × 25/40 °C × MG/PVSmoothing/PowerQuality 패턴 × 5~30 weeks | Multi-cell + multi-pattern cross-validation |
| 모듈 | KIER module CH1 | NCM Pouch 14S 모듈 | 25 °C × ~3000 cycle (Pattern 분할 파일 포함) | **Cross-scale (cell → module) 외삽 검증** |
| (옵션) NASA PCoE | 18650 NCA/NMC | 외부 18650 | 시간 남고 reviewer 추가 어필 필요 시 | Form-factor 다른 별도 contribution으로만 |

**의도적 제외**:
- **18650_SOH** (자체 18650): Pouch 폼팩터 일관성 유지를 위해 사용자 결정으로 제외 (2026-04-23).
- CALCE / MIT-Stanford: 미드티어 노선에선 불필요. 자체 Pouch 셀+모듈로 충분.
- −15 °C, 45 °C 데이터는 **본 폴더에 존재하지 않음** → 외삽 실험은 가용 범위(0/25/40/50 °C) 내에서만.
- Cell 데이터★★★★★ (2018년 dithering): 다른 셀 시리즈로 추정, 사용자 결정으로 제외.

### 5.2 Split 프로토콜 (실데이터 가용 범위)

- **Cell-level split**: main_cell 내에서는 ch2 ↔ ch7 단 두 셀 (단, 두 셀이 서로 다른 온도라 cell+temp 혼합 외삽이 됨). 진정한 cell-level은 pouch_SOH 다채널에서 구현.
- **Condition-level split** (main_cell): 학습 = (25 °C, cycle 100~1500) ∪ (50 °C, cycle 100~1500), 테스트 = (25 °C, cycle 2000~3000) ∪ (50 °C, cycle 2500~3800) — **unseen aging extrapolation**.
- **Temperature-level split**: 학습 = main_cell (25/50 °C), 테스트 = (0 °C, cycle 0~101). 0 °C 데이터는 다른 셀이라 cell+temp 이중 외삽임을 표기. **Supplementary**: 5/10/15 °C Pattern 데이터에서 SOC RMSE 만 추가 보고 (SOH 는 RPT 부재로 불가) — wide thermal envelope (0~50 °C) 일반화.
- **Time-ordered split**: 단일 파일 내 시점 분할 — 학습 70% / 테스트 30%, 시점 누설 없음.
- **Cross-pattern split** (pouch_SOH): 학습 = MG + PVSmoothing, 테스트 = PowerQuality 같은 unseen 주행패턴.
- **Cross-scale split** (cell → module): 학습 = main_cell ch2/ch7 전체, 테스트 = module CH1 (cell-equivalent 또는 module-direct 두 시나리오). **본 논문의 두 번째 contribution 축**.

### 5.3 Baseline 11종 (P1에서 모두 구현)

| 분류 | 모델 |
|---|---|
| 비-필터 | Ah Counting, OCV Look-up |
| 정수 필터 | EKF, UKF, DEKF, DUKF |
| 적응 필터 | AEKF (innovation-based) |
| 데이터 | LSTM-Joint, TCN-Multi-task, Transformer-Multi-task |
| SOH 회귀 | GPR (Gaussian Process Regression) |

### 5.4 평가 메트릭

- **정확도**: SOC/SOH 각각 RMSE, MAE, MAX absolute error.
- **강건성**: 초기 SOC 섭동 (±10/20/30%) 하의 수렴 시간·잔차.
- **외삽**: 학습에 없는 (T, cycle) 조합에서의 RMSE 증가율.
- **UQ**: PICP_95 (95% 예측구간 적중률, 이상값 0.95), NLL.
- **임베디드성**: 추론 시간 µs/step, 메모리 KB.
